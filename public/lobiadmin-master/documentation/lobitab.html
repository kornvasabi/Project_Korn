LobiTab

LobiTab is jquery plugin for bootstrap tabs exclusively written for LobiAdmin. Using it you can add/remove/edit/sort tabs.

Basic Usage

LobiTab initialization is simple. Just create correct markup of bootstrap tabs and call plugin.

<div id="lobitab-demo">
    <!-- Nav tabs -->
    <ul class="nav nav-tabs" role="tablist">
        <li class="active"><a href="#home1" role="tab" data-toggle="tab">Home</a></li>
        <li><a href="#profile1" role="tab" data-toggle="tab">Profile</a></li>
        <li><a href="#messages1" role="tab" data-toggle="tab">Messages</a></li>
        <li><a href="#settings1" role="tab" data-toggle="tab">Settings</a></li>
    </ul>

    <!-- Tab panes -->
    <div class="tab-content">
        <div class="tab-pane active" id="home1">Home</div>
        <div class="tab-pane" id="profile1">Profile</div>
        <div class="tab-pane" id="messages1">Messages</div>
        <div class="tab-pane" id="settings1">Settings</div>
    </div>

</div>
<script>
    $(function(){
        $('#lobitab-demo').lobiTab();
    });
</script>

And you will get the following result.

Methods

Methods can be called in two ways.

1. You can get the LobiTab instance and call the method

//get the LobiTab instance
var instance = $('#lobitab-demo').data('lobiTab');
//call the methods
instance.enableClose();
instance.addNewTab();
instance.setTitle("#example", "Custom title");
var title = instance.getTitle("#example");
//method chaining is also available
instance.enableClose(400, 400)
    .addNewTab("New tab title", "new tab content", "newTabId")
    .setTitle("newTabId", "Another title");

2. Or you can call the method directly

$('#lobitab-demo').lobiTab('enableClose');
$('#lobitab-demo').lobiTab("addNewTab", "New tab title", "new tab content", "newTabId");
var title = $('#lobitab-demo').lobiTab("getTitle", "#example");

Events

Using events

$('#lobitab-demo').on('init.lobiTab', function(ev, lobiTab){
    window.console.log("init event called");
});
$('#lobitab-demo').on('added.lobiTab', function(ev, lobiTab, a){
    window.console.log("After tab added", lobiTab);
});
$('#lobitab-demo').on('startEditing.lobiTab', function(ev, lobiTab, a){
    window.console.log("Start title editing", ev, lobiTab);
});
$('#lobitab-demo').on('finishEditing.lobiTab', function(ev, lobiTab, a){
    window.console.log("Finish editing", ev, lobiTab, a);
});

Ajaxed tabs

Load content from ajax whenever tab is shown. Or you can load once when plugin is initialized. Or you can load once when tab is shown at first time.

As this example is based on ajax it will not work unless you put this file on server.

  • Messages 
<div id="lobitab-demo-ajax">
    <!-- Nav tabs -->
    <ul class="nav nav-tabs" role="tablist">
        <li class="active">
            <a href="#lobitab-demo-ajax .home" role="tab" data-toggle="tab" data-load-url="pages/demo/home.html">
                <i class="fa fa-home"></i> <span class="lobitab-title">Home</span>
            </a>
        </li>
        <li>
            <a href="#lobitab-demo-ajax .profile" role="tab" data-toggle="tab">
                <i class="fa fa-user"></i> <span class="lobitab-title">Profile</span>
            </a>
        </li>
        <li>
            <a href="#lobitab-demo-ajax .messages" role="tab" data-toggle="tab" data-load-url="pages/demo/messages.html">
                <i class="fa fa-envelope"></i> <span class="lobitab-title">Messages</span>
            </a>
        </li>
    </ul>

    <!-- Tab panes -->
    <div class="tab-content">
        <div class="tab-pane active home">Home</div>
        <div class="tab-pane profile">Profile</div>
        <div class="tab-pane messages">Messages</div>
    </div>
</div>        
<script>

  $('#lobitab-demo-ajax').lobiTab({
        loadByAjax: true,
        lazyLoad: true,
        loadOnce: false
    });
</script>
